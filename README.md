# Simple-Ecomerce

MVP end‑to‑end pequeño que refleje tu criterio técnico. Incluye auth básica, catálogo (Stores, Products, StoreProducts) y vistas para operarlo

## Production Considerations

This project uses a local PostgreSQL database for simplicity during development.
In a production environment, we would:

- Use environment-specific configurations
- Implement database migrations
- Use cloud PostgreSQL with proper backups
- Set up CI/CD pipelines
